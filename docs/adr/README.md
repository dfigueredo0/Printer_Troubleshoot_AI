# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the RuFlo / Claude Flow v3 project, following the [MADR](https://adr.github.io/madr/) format.

ADRs are auto-checked by `.claude/helpers/adr-compliance.sh` every 15 minutes during active development.

## Status legend

- **Accepted** — decision is in force
- **Superseded** — replaced by a later ADR (link provided)
- **Deprecated** — no longer relevant
