# ADR-001: agentic-flow as core foundation

## Status

Accepted

## Context

We need a stable, well-tested base for agent lifecycle, memory, and task execution rather than reimplementing these primitives.

## Decision

Use `agentic-flow` as the core foundation package. Claude Flow v3 is built as a specialized extension on top of it rather than a parallel implementation.

## Consequences

### Positive

Eliminates ~10,000 lines of duplicate code. Ensures we inherit upstream improvements automatically. Reduces maintenance surface.

### Risks and mitigations

Tight coupling to agentic-flow release cadence. Mitigated by pinning versions and maintaining a thin adapter layer.
